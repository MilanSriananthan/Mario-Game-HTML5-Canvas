# MARIO Game

Utilized HTML5 Canvas and Javascript in order to create a mario type game format

## Functions:

- Player Creation
- Gravity
- Player Movement
- Platforms
- Scrolling the background
- Win and Lose Scenario
- Image Platforms
- Parallax scrolls
- Death pits
- Level Creation
